# 프로젝트 계획

## 목표
- 안드로이드 앱을 Kotlin + Jetpack Compose로 구성
- 입대일/전역일을 설정하면 현재 복무 진행률을 계산
- 진행률을 `소수점 5자리`까지 한글로 표시
- 같은 값을 4:1 비율 이미지로 생성
- 이미지 아래에 `복사`, `저장` 버튼 제공
- 인스타 스토리 스티커처럼 쓰기 좋게 `투명 배경 PNG` 기준으로 구현

## 구현 방향
- 오버엔지니어링 없이 `단일 Activity + Compose 화면`으로 구성
- 상태는 화면 내부 `rememberSaveable` 중심으로 단순 관리
- 날짜 입력은 안드로이드 기본 `DatePickerDialog` 사용
- 이미지 생성은 `Bitmap + Canvas`로 직접 그림
- 저장은 `MediaStore` 사용
- 복사는 생성한 PNG를 캐시 파일로 만든 뒤 `FileProvider` URI를 클립보드에 넣는 방식으로 구현

## 화면 구성
1. 상단 제목/설명
2. 입대일 선택 버튼
3. 전역일 선택 버튼
4. 계산된 진행률 텍스트
5. 진행률 바 미리보기
6. 4:1 이미지 미리보기
7. 복사 / 저장 버튼

## 계산 규칙
- 전체 기간: `전역일 - 입대일`
- 경과 기간: `오늘 - 입대일`
- 진행률: `경과 / 전체 * 100`
- 범위는 `0% ~ 100%`로 고정
- 표시 형식은 소수점 다섯째 자리까지

## 검증
- 변경 Kotlin 파일 진단 확인
- 가능하면 Gradle 빌드 실행
- 현재 환경에 Android SDK/Gradle 제약이 있으면 그 사실을 명시
